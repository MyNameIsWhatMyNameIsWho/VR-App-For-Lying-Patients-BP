## Overview

This project is a set of interactive VR therapeutic modules designed for bedridden seniors, developed for my Bachelor thesis at Czech Technical University in Prague.  
It extends and improves an existing Unity-based VR rehabilitation application, targeting cognitive and physical stimulation in a user-centered, accessible way.

**Platform:** Meta Quest 2/3  
**Engine:** Unity (C#)  
**Status:** Functional prototype, tested in a real senior care facility.

---

## Features

- **Two new VR game modules**: Each with original mechanics focused on accessibility and engagement for seniors.
- **Audio tutorials**: Step-by-step voice guidance for ease of use, even for users unfamiliar with technology.
- **User-Centered Design (UCD)**: Iteratively developed with feedback from senior patients and care staff.
- **Legacy code integration**: Built on top of a previous Master's thesis application, with full compatibility and code refactoring.
- **Optimized for Meta Quest**: Seamless performance and VR interaction on Quest 2/3.
